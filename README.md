
# Bunker

A attendance manager for students. 

For now it is client side web app only

## Tech Stack

**Client:** React



## Run Locally

Clone the project

```bash
  git clone https://github.com/suraj-kush/bunker.git
```

Go to the project directory

```bash
  cd bunker
```

Install dependencies

```bash
  npm install
```

Run the app

```bash
  npm run start
```
